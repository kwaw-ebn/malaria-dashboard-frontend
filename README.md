> A modern, interactive dashboard for predicting malaria risk using AI and weather data.  

## 🔥 Features

- 🌡️ Input rainfall, temperature, and humidity to predict malaria risk.
- 🚨 Risk prediction with clear classification: High 🔴, Medium 🟡, Low 🟢.
- 📈 Trend forecasting charts with **Chart.js**.
- 🗺️ Interactive Nigeria malaria risk map with **Plotly.js** integration.
- ⚡ Responsive, modern design using **Tailwind CSS**.
- 🔗 Connects to a backend API hosted on [Render](https://backend-3x5i.onrender.com/).

## 💻 Tech Stack

- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Charts & Visualization:** Chart.js, Plotly.js  
- **Backend API:** FastAPI (hosted on Render)
